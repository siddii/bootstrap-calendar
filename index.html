A simple calendar plugin for the awesome Bootstrap framework

Example

Here is a basic use-case of this plugin. It adds notes to any date on the calendar. Currently, this example saves data in the localStorage.

Markup

By adding the following markup, you will get the calendar rendered like the one shown above. It internally leverages Bootstrap's Carousel plugin for the layout & transitions with a default template. However, you can configure it to some other template based on your needs. 

<div data-toggle="calendar"></div>
